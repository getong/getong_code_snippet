# elixir node
## a node
```
iex --sname a
1> Node.self()
:b@DebianLighten

```
## b node
```
iex --sname b
iex(b@DebianLighten)1> Node.self()

iex(b@DebianLighten)2>

```
## connection
```
iex(b@DebianLighten)2> Node.connect :a@DebianLighten
true
```
