```
 {initial_call,{erlang,apply,2}},

```

##current_function
当前函数

##initial_call
初始调用函数

##status
当前进程运行状态

##message_queue_len
消息队列长度

##messages
进程收到的消息

##links
链接进程

##dictionary
进程字典

##trap_exit
捕获进程退出标志

##error_handler
错误处理器

##priority
进程优先级

##group_leader
io输出进程

##total_heap_size
总的堆的大小

##heap_size
堆大小

##stack_size
栈大小

##garbage_collection
进程垃圾回收选项

##suspending
被其挂起的所有进程信息
