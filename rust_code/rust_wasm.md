# rust wasm

## install wasm-pack

```
curl https://rustwasm.github.io/wasm-pack/installer/init.sh -sSf |  sh

## new a wasm pack project
wasm cargo generate --git https://github.com/rustwasm/wasm-pack-template

## change directory
cd $project

## build wasm
wasm-pack build
```
