* vue installation

** install via npm

#+begin_src shell
npm install -g @vue/cli
#+end_src

** install via homebrew

#+begin_src shell
brew install vue-cli
#+end_src


** create project example

#+begin_src shell
vue create app
cd app
yarn serve
#+end_src

** install vue language server

#+begin_src shell
npm i @volar/vue-language-server
npm install -g emmet-ls
#+end_src
